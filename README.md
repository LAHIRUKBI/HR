## Technology Stack

**Frontend:**
- React.js
- Redux (State management)
- React Router (Navigation)
- Axios (HTTP client)
- Tailwind CSS (Styling)

- Node.js
- Express.js
- MongoDB (Database)
- Mongoose (ODM)
- JSON Web Tokens (Authentication)
- Bcrypt (Password hashing)
